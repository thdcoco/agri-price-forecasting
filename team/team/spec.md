# 전력수요 SARIMA vs SARIMAX 분석 — 작업 지시서

## 목표
한국 전력수요를 **SARIMA(외생변수 없음)** 와 **SARIMAX(기온 외생변수)** 두 모델로
분석·비교하고, **2026년 상반기 수요를 예측**한다.
핵심 메시지: *"전력수요는 기온에 강하게 반응하며, 기온을 외생변수로 넣은 SARIMAX가
SARIMA보다 변동을 잘 설명한다."*

---

## 입력 데이터

### 1) 전력수요 (전국, 시간별)
- 파일: 2013~2025 + 2024 연도별 CSV
- 컬럼: `날짜, 1시, 2시, ... , 24시`
- 처리: **24개 시간값을 일평균** 내어 일별 수요(`load`)로 변환
- 커버리지 확인: 2013~2026 연속 (2024 포함, 구멍 없어야 함)

### 2) 기온 (서울, 일별) — 외생변수 X
- 파일: `ta_...1016.csv` (2013-01-01 ~ 2025-12-31)
- 컬럼: `날짜, 지점, 평균기온(℃), 최저기온(℃), 최고기온(℃)`
- **인코딩: cp949** (euc-kr)
- 데이터는 헤더 메타 6줄 이후 시작, **각 데이터 행 앞에 탭 문자** 있음 → strip 필요

### 3) 2026 기온 (미래 외생변수)
- 파일: `ta_...1034.csv` (2026-01-01 ~ 2026-06-01)
- 위와 동일 포맷. 예측 구간의 exog로 사용.

> 참고: 기온은 서울(지점 108)만 사용. 수요는 전국이므로
> "서울 기온을 전국 대표값으로 사용"이라는 가정을 발표에 명시할 것.

---

## 전처리
1. 연도별 수요 CSV 전부 병합 → 일평균 수요 `load` 생성
2. **단위를 GW로 변환** (MW → GW, ÷1000). SARIMAX 최적화 수렴 안정성 위함
3. 기온 파일과 `날짜` 기준 merge → `date, load, temp_avg`
4. 외생변수 생성 (기온의 U자형 효과를 선형화):
   - 냉방도일 `CDD = max(평균기온 - 24, 0)`
   - 난방도일 `HDD = max(18 - 평균기온, 0)`
5. (선택) 명절(설·추석) 더미 변수 추가 → "달력 효과"까지 설명

---

## 분석
1. **ADF 정상성 검정** → 차분 차수 `d` 결정 (원계열/1차차분/계절차분 비교)
2. **ACF/PACF** 로 `(p, q)` 추정. 계절성 주기 `s=7` (주간 패턴)
3. 두 모델 적합:
   - **모델 A — SARIMA**: `order=(p,d,q)`, `seasonal_order=(P,D,Q,7)`, 외생변수 없음
   - **모델 B — SARIMAX**: 동일 차수 + `exog=[CDD, HDD]`
4. **AIC 비교** + 검증구간 RMSE / MAPE
5. SARIMAX **외생변수 계수와 p-value** 출력 → "기온 1도일당 수요 변화량" 해석

---

## 예측 (2026 상반기)
1. 전체(2013~2025) 데이터로 재학습
2. **2026-01-01 ~ 2026-06-01 수요 예측**
   - SARIMAX: 실제 2026 기온으로 만든 CDD/HDD를 `exog`로 입력
   - SARIMA: 외생변수 없이 과거 패턴만 연장
3. 그래프: SARIMA 예측선 vs SARIMAX 예측선 + SARIMAX 95% 신뢰구간
   - 대비 포인트: SARIMA는 평탄/패턴연장, SARIMAX는 2026 기온 굴곡(1월 한파 등)을 추종

---

## 산출물
- 정상성 검정 결과 표
- 모델 비교 표 (AIC / RMSE / MAPE)
- 외생변수 계수·유의성 표
- 그래프: ① 전체 시계열 ② 검증구간 확대 ③ 2026 예측

## 주의점 (시행착오 방지)
- 수요 값이 수만 단위(MW)면 SARIMAX 최적화가 **수렴 실패** 가능 → **GW 단위** 권장
- 검증으로 모델 우위를 보이려면 **기온 급변 구간(여름 피크 등)** 을 검증셋으로 잡아야 차이가 드러남
- matplotlib **한글 폰트 깨짐** 주의 → 나눔폰트 등록 (`fonts-nanum`)
- 기온 CSV는 cp949 인코딩 + 행 앞 탭 → `pd.read_csv(..., encoding='cp949')` 후 공백 strip
- 2026 수요 실측이 없으므로 2026 구간은 **예측만** 표시 (정확도 검증 아님)
